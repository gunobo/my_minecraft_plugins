# my_minecraft_plugins

## treecut: 벌목 커스텀 인첸트 플러그인
- `/treecut <1~3>` 으로 벌목 인첸트북 획득가능
- 레벨1: 나무가 한번에 캐질 확률:: 15%
- 레벨2: 나무가 한번에 캐질 확률:: 20%
- 레벨3: 나무가 한번에 캐질 확률:: 25%

## OreExploration: 광맥탐지 커스텀 인첸트 플러그인
- `/oreexploration` 라는 명령어로 인첸트북 획득가능
- 곡괭이에만 인첸트 가능
- 광물근처 5블럭→ 블럭 테두리만 발광
- 쿨타임:: 10초

## CropGrowth: 농작물 성장 커스텀 인첸트 플러그인
-`/cropgrow` 명령어로 인첸트북 획득 가능
- 괭이에만 인첸트 가능
- 농작물 성장: 30% 확률로 수확 시 씨앗이 즉시 성장.

## MagicProtection: 마법 보호 커스텀 인첸트 플러그인
- `/magicprotection` 명령어로 획득가능
- 투구, 흉갑에 적용가능
- 포션*마법 피해감소: 15%

## NightVisionArmor: 야간시야 커스텀 인첸트 플러그인
- `/visionnight` 명령어로 인첸트북 획득가능
- 투구에만 인첸트 가능
- 인첸트된 투구 작창시 야간투시 받음(무한)
- 투구 장착 해제시 야간투시 clear

## SpeedBoost: 속도강화 커스텀 인첸트 플러그인
- `/speedenchant <1~3>` 명령어로 커스텀 인첸트북  획득가능
- 레벨1: 기존 속도의 5% 빨라짐
- 레벨2: 기존속도의 15%빨라짐
- 레벨3: 기존속도의 20%빨라짐
